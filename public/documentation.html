Description Endpoint URL HTTP Method Request body data format Response body data format
Return a list of all movies to the user /movies GET - A JSON object holding data about all the movies
Return data (description, genre, director, image URL, feature or not) about a single movie by title /movies/:Title GET - A JSON object holding data about a movie
Return data about a genre (description) by name /movies/genre/:Name GET - A JSON object holding data about a genre
Return data about a director (bio, birth year, death year) by name /movies/director/:Name GET - A JSON object holding data about a director
Allow new users to register /users POST { "Username": "Username", "Password": "Password", "Email": "user@email.com", "Birthday": "09/09/1990" } A JSON object holding data about the newly added user
Allow users to update their info base on username /users/:Username PUT { "Username": "Username", "Password": "NewPassword", "Email": "user@email.com", "Birthday": "09/09/1991" } A JSON object holding data about the modified user
Allow users to add a movie to their list of favorites /users/:Username/movies/:MovieID POST - A text message indicating the movie was added to the user's favorite list
Allow users to remove a movie from their list of favorites /users/:Username/movies/:MovieID DELETE - A text message indicating the movie was removed from the user's favorite list
Allow exisitng users to deregister /users/:Username DELETE - A text message indicating the user has been deleted
