Documentation page

This is the documentation page of the movie API for Achievement 2

Description Endpoint URL HTTP Method Request body data format Response body data format
Return a list of all movies to the user /movies GET None A JSON object holding data about all the movies
Return data (description, genre, director, image URL, feature or not) about a single movie by title /movies/:title GET None A JSON object holding data about a movie
Return data about a genre (description) by name /movies/genre/:genreName GET None A JSON object holding data about a genre
Return data about a director (bio, birth year, death year) by name /movies/directors/:directorName GET None A JSON object holding data about a director
Allow new users to register /users POST { "name": "Firstname Lastname" } A JSON object holding data about the newly added user
Allow users to update their user name /users/:id PUT { "name": "New Name" } A JSON object holding data about the modified user
Allow users to add a movie to their list of favorites /users/:id/:movieTitle POST None A text message indicating the movie was added
Allow users to remove a movie from their list of favorites /users/:id/:movieTitle DELETE None A text message indicating the movie was removed
Allow exisitng users to deregister /users/:id DELETE None A text message indicating the user has been deleted
